## How to Install

我们已经准备好了所有的package.json，只需要clone下来，在安装node后，文件根目录下输入

```shell
cd react_component
npm install --dependencies
```

安装完依赖后

```shell
./run.sh
```

打开浏览器，进入

```
127.0.0.1:3000
```
